# Envelope Budgeting App

## Monorepo Structure

This is a monorepo for the Envelope Budgeting App, a personal finance tool designed to help users manage their money using the envelope budgeting methodology.

### Directory Structure

- `web-frontend/`: React-based web application
- `backend/`: Node.js/Express API server
- `ios-frontend/`: Swift-based iOS application
- `android-frontend/`: Kotlin-based Android application

## Getting Started

Each subdirectory contains its own README with specific setup instructions.

## Development Workflow

- The `main` branch contains stable code
- Development occurs in feature branches:
  - `web-dev`: Web frontend development
  - `backend-dev`: Backend API development
  - `ios-dev`: iOS app development
  - `android-dev`: Android app development
